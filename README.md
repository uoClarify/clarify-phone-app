## Welcome to the Clarify Phone App Repository

### Process to make an expo app:
    - npm install -g expo-cli
    - expo init MyExpoApp
    It will prompt you with choices, I choose typscript
    - cd MyExpoApp
    - run 'expo start'

    -press w for browser access
